# mushroom-game

* 2D game made using Godot (GDScript)
* Graphics also done by tomual on Aseprite
* 4 maps, 2 monsters, 2 NPCs
* Item drops, player inventory
* Level-up NPC
* Weapon Upgrade NPC
* Following enemies, miniboss

Download here:
https://github.com/tomual/mushroom-game/releases/tag/1
